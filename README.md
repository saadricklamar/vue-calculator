## vue-calculator

This is a project I decided to take up on my spare time to keep my Vue.js skills sharp and up to date. While React calculators may be all the craze, Vue calculators are the next cool thing. 

## Teck Stack

* JavaScript
* Vue.js
* Vue Test Utilities
* Jest/Enzyme
